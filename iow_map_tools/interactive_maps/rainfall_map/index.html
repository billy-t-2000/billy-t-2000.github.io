Features

ArcGIS Maps SDK for JavaScript API

  • ArcGIS Server Comptibility
  • Dynamic Tools
  • Informative Popups
  • ArcGIS Pro Integration
  • Multiple Online Formats

ArcGIS Server Comptibility

Layers can be hosted in ArcGIS Server and easily brought into the application.

Dynamic Tools

Tools such as a layer changer and legend can easily be added to the tool.

Informative Popups

Many different streams of information can be added, such as tables, charts, and text. In this case, clicking on the kriging contour regions brings up a host of information.

ArcGIS Pro Integration

Data can be processed in ArcGIS Pro, using the myriad of tools available there, before being easily uploaded to the ArcGIS Server. In this case, many tools were used to calculate various features of the rainfall kriging contours.

Multiple Online Formats

Many different formats can be imported from the ArcGIS Server. In this map, the Feature Access REST API was used for most of the layers, but the LIDAR Elevation layer, which is a raster, is imported in a WMS format.
