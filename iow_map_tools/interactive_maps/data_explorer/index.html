Features

ArcGIS Web AppBuilder

  • Simple Building
  • Dynamic Mapping Sources
  • ArcGIS Hosting
  • Powerful Tools
  • Dynamic Information

Simple Building

The ArcGIS Online and ArcGIS Web AppBuilder platforms make it quick and simple to build a powerful web mapping tool.

Dynamic Mapping Sources

Dynamic information layers from organiations around the world, such as NOAA satellite data, can easily be brought into the map as layers.

ArcGIS Hosting

All data and code is hosted on the ArcGIS Online servers, keeping your web domain simpler.

Powerful Tools

The app builder gives you access to many powerful tools, such as search, directions and swipe, which can be added into the platform.

Dynamic Information

All layers have pop-up information.
