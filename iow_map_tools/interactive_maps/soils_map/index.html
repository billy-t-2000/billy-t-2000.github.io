Features

Leaflet Maps API

  • GeoServer Compatibility
  • Open Source
  • Responsive
  • Dynamic Legend and Information

GeoServer Compatibility

The data is taken from GeoServer in the form of a GeoJSON, sent via the JSONP protocol.

Open Source

The GeoJSON is an open-source format, and the Leaflet Maps API is an open-source map builder.

Responsive

The tool is quick and responsive, updating the appearance of layers quickly as the mouse moves and clicks.

Dynamic Legend and Information

The legend and information pop-up are created and shown dynamically by the API code.
